# ⏰ Demo Clock - 实时时钟展示页

🚀 **在线体验**: https://demo-clock.vercel.app/

---

## ✨ 功能亮点

### 🎯 精准计时
- **实时显示** - 每秒自动刷新，毫秒级精准
- **日期星期** - 一目了然，告别忘记今天是周几

### 🌍 多时区支持 (v2.0 新功能)
- **一键切换** - 本地/纽约/伦敦/东京 4个时区
- **实时换算** - 自动计算时差，准确显示

### 🎨 精美设计
- **深色渐变背景** - 护眼舒适，适合长时间展示
- **毛玻璃效果** - 现代感十足的视觉体验
- **发光字体** - 科技感拉满
- **双主题切换** - 🌙深色/☀️浅色模式一键切换 (v2.0)

### 💡 创新交互
- **秒针指示器** - 5点动态光点，时间流逝可视化
- **响应式布局** - PC、平板、手机完美适配
- **全屏模式** - 双击页面进入沉浸体验，ESC退出 (v2.0)
- **键盘快捷键** - T键切换主题，F键切换全屏 (v2.0)

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/hoopy1987/demo-clock.git

# 进入目录
cd demo-clock

# 本地打开
open index.html
```

### 快捷键说明
| 按键 | 功能 |
|------|------|
| `T` | 切换深色/浅色主题 |
| `F` | 切换全屏模式 |
| `ESC` | 退出全屏 |
| `双击` | 进入全屏 |
